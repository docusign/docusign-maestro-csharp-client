Detailed Description

Account model with the following properties: accountId: the account ID GUID. isDefault: whether this is the default account, when the user has access to multiple accounts. accountName: the human-readable name of the account. baseUri: the base URI associated with this account. It also tells which Docusign data center the account is hosted on.
